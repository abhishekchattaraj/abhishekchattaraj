## 📂 Featured Projects

### 🏥 ChronoPath Diagnostics: Operational Performance & TAT Analysis  
🔗 [View Project](https://github.com/abhishekchattaraj/ChronoPath-Diagnostics-Dashboard)  

**Tools:** Power BI, Python, Excel  
- Built an end-to-end operational dashboard to analyze **Turnaround Time (TAT)** for 20,000 diagnostic tests across 5 zonal hubs.  
- Identified **SLA breach patterns** and bottlenecks across logistics, lab processing, and reporting stages.  
- Simulated realistic data using Python with **seasonality, routing errors, and test complexity**.  
- Designed a **Star Schema data model** and interactive drill-down dashboards for executive and operational views.  
- Key Insight: **Lab Processing (Stage 2)** is the primary bottleneck, especially for complex tests like COVID RT-PCR and MRI.
